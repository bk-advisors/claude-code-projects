# Common AI Writing Tells — Reference Guide

A checklist for spotting (and avoiding) the giveaways that AI wrote something. Compiled from multiple sources for use when editing AI-assisted drafts.

## 1. Overused "AI Words"

These words are rarely used by humans in casual or professional writing but appear constantly in AI output:

| AI Word | Human Alternative |
|---------|-------------------|
| delve | dig into, look at, explore |
| tapestry | mix, combination, web |
| multifaceted | complex, layered, complicated |
| landscape (as metaphor) | world, field, space |
| leverage | use, build on |
| harness | use, tap into |
| utilize | use |
| streamline | simplify, speed up |
| underscore | highlight, show |
| pivotal | key, important, critical |
| robust | strong, solid |
| innovative | new, fresh |
| seamless | smooth |
| cutting-edge | modern, new, latest |
| realm | area, field, world |
| synergy | teamwork, collaboration |
| testament | proof, sign, evidence |
| endeavour | effort, attempt, try |
| illuminate | show, reveal, clarify |
| navigate (metaphorical) | deal with, work through, figure out |
| embark | start, begin |
| vibrant | lively, active |
| compelling | strong, interesting, convincing |
| nuanced | subtle, detailed |
| foster | build, encourage, grow |

## 2. Overused Phrases

| AI Phrase | What to write instead |
|-----------|----------------------|
| "In today's ever-evolving..." | Drop it. Just start. |
| "It's important to note that..." | Just state the thing. |
| "In summary / In conclusion / In essence" | Skip the signpost or use a short transition. |
| "At its core" | Drop it or say "basically" / "really" |
| "That being said" | "But" / "Still" / "Though" |
| "This is where X comes in" | Just introduce X. |
| "Let's delve into" | "Here's how" / "Let's look at" |
| "Not just X — it's Y" | Use once at most. AI loves this pattern. |
| "stands as a testament to" | "shows" / "proves" |
| "Whether you're a X or a Y" | Drop the false dichotomy. |
| "From X to Y" (as a scope claim) | Be specific about what you actually cover. |

## 3. Structural Tells

- **Rule of three everywhere.** AI defaults to three-item lists. Humans vary — sometimes two, sometimes four, sometimes one.
- **Perfectly parallel structure.** Every bullet starts the same way, every sentence follows the same rhythm. Humans are messier.
- **Every section has the same length.** AI produces suspiciously uniform paragraphs and sections. Real writing is uneven.
- **Excessive em dashes.** AI sprinkles em dashes (—) like confetti. One or two per page is human. Five per paragraph is AI.
- **Bold + colon pattern.** "**Key insight:** Here's the thing." AI does this constantly. Use it sparingly.
- **Summary at the end of every section.** Humans don't always wrap up neatly. Sometimes you just... move on.
- **Headers on everything.** AI over-structures. Not every paragraph needs a subheading.
- **Numbered lists for processes.** AI defaults to 1-2-3 steps. Sometimes a process is better described in prose.

## 4. Tone Tells

- **Relentlessly positive.** AI avoids negativity, hedges, and uncertainty. Real people doubt things, get frustrated, and admit when something sucks.
- **Too smooth.** No rough edges, no tangents, no personality. Every transition is polished. Real writing has texture.
- **Over-explains.** AI defines terms nobody asked about. If your audience knows what a heatmap is, don't explain it.
- **False enthusiasm.** "This is a game-changer!" "The results were remarkable!" Humans are more understated.
- **No personal quirks.** No humor, no self-deprecation, no odd references. AI stays on-topic like a textbook.
- **Empathy performance.** "I understand how challenging this can be." AI says this without having experienced anything.
- **Hedging then asserting.** "While there are many approaches, the most effective one is..." — a formulaic AI hedge.

## 5. Content Tells

- **No specific names, dates, or places** (unless asked). AI generalizes. Humans reference specific people, specific conversations, specific Tuesday afternoons.
- **Metaphors that don't quite work.** AI generates metaphors that are technically coherent but feel off. "Like a compass in a storm of data" — nobody actually talks like that.
- **Quoting without sourcing.** AI invents plausible-sounding quotes or attributes vague wisdom to unnamed experts.
- **Even-handed to a fault.** AI presents "both sides" when a human with experience would just take a position.
- **No contradictions or messiness.** Real thinking involves changing your mind, qualifying earlier statements, and admitting exceptions. AI moves in a straight line.

## 6. Quick Self-Edit Checklist

Before publishing AI-assisted writing, check for:

- [ ] Did I use "delve," "tapestry," "landscape," "multifaceted," or "leverage"? Replace them.
- [ ] Does every section end with a tidy summary? Cut some of them.
- [ ] Are all my lists groups of three? Vary the count.
- [ ] Is the tone relentlessly positive? Add some honest doubt or frustration.
- [ ] Did I over-explain concepts my audience already knows? Trust the reader.
- [ ] Are there specific names, places, and dates? Add real details from memory.
- [ ] Does it sound like *me* or like a well-meaning textbook? Read it aloud.
- [ ] Count the em dashes. If there are more than 3-4 per page, cut some.
- [ ] Is there a single moment of genuine surprise, humor, or vulnerability? If not, add one.
