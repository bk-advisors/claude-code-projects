Chart Reengineering 03 — Measles Vaccination Heatmap for Africa

Real WHO data showing how mass immunization campaigns transformed a continent

About This Lesson

Lessons 01 and 02 used synthetic MNH data and the original US measles dataset. This lesson tells the real story of measles vaccination in Africa using WHO surveillance data and World Bank population estimates — 46 countries across 44 years.

The landmark year is 2001, when the WHO/UNICEF Measles & Rubella Initiative launched targeted supplementary immunization activities (SIAs) across Africa. Within a decade, reported measles cases dropped by over 90% in most countries.

Data Sources

The pre-processed CSV (data/africa_measles.csv) was built from two public APIs:

Source Indicator What It Provides
WHO Global Health Observatory WHS3_62 Measles reported cases by country, 1980-2023
World Bank Open Data SP.POP.TOTL Total population by country, 1980-2023

To regenerate the data from scratch, run source("../data/prep_africa_measles.R") (requires jsonlite).

The Answer

A publication-ready heatmap of measles cases per million across 46 African countries from 1980-2023, with a reference line marking the 2001 WHO/UNICEF Measles Initiative:

library(ggplot2)
library(dplyr)
source("../../synthesized-lessons/_common/theme_bka.R")

# ---- Load pre-processed data ----
# Built from WHO GHO API (cases) + World Bank API (population)
# To regenerate: source("../data/prep_africa_measles.R")
africa_measles <- read.csv("../data/africa_measles.csv")

# ---- Sort countries by average pre-2001 case rate (highest at top) ----
# See Lesson 1.2 (Aesthetics) — factor() with custom levels for y-axis ordering
country_order <- africa_measles %>%
  filter(year < 2001) %>%
  group_by(country) %>%
  summarise(avg_rate = mean(cases_per_million, na.rm = TRUE), .groups = "drop") %>%
  arrange(avg_rate) %>%
  pull(country)

africa_measles <- africa_measles %>%
  mutate(country = factor(country, levels = country_order))

# ---- BKA multi-stop gradient ----
# See Lesson 2.4 (Palettes) — colorRampPalette() creates smooth multi-stop gradients
# Asymmetric split: 15 steps for low-mid range, 5 for high = visual urgency at top end
measles_gradient <- c(
  colorRampPalette(c("#ACCBF9", "#005CB9", "#83BD00", "#F8A623"))(15),
  colorRampPalette(c("#F8A623", "#FA7650", "#E24A3F"))(5)
)

# ---- The heatmap ----
ggplot(africa_measles, aes(x = year, y = country, fill = cases_per_million)) +
  # See Lesson 1.4 (Geometries) — geom_tile() with white borders
  geom_tile(colour = "white", linewidth = 0.4,
            width = 0.9, height = 0.9) +    # Tufte: white space between tiles
  # See Lesson 2.4 (Palettes) — scale_fill_gradientn() for multi-stop gradients
  scale_fill_gradientn(
    colours = measles_gradient,
    limits = c(0, 5000),
    breaks = seq(0, 5000, by = 1000),
    labels = c("0", "1K", "2K", "3K", "4K", "5K"),
    na.value = "#F5F5F5",
    name = "Reported cases per million population",
    oob = scales::squish,    # Squish outliers beyond 5000 to the max color
    # See Lesson 2.5 (Legend Styling) — guide_colourbar() for continuous legends
    guide = guide_colourbar(
      barwidth = 15, barheight = 0.6,
      title.position = "top",
      title.hjust = 0.5,
      ticks = TRUE, nbin = 50
    )
  ) +
  # See Lesson 1.7 (Scales) — expand = c(0,0) removes padding
  scale_x_continuous(expand = c(0, 0),
                     breaks = seq(1980, 2020, by = 5)) +
  # See Lesson 3.2 (Annotations) — geom_vline() marks a policy milestone
  geom_vline(xintercept = 2001, color = bka_colors$title_dark,
             linewidth = 1.2, alpha = 0.8) +    # Cairo: annotations are essential, not optional
  # See Lesson 3.2 (Annotations) — annotate("text") for floating labels
  annotate("text", label = "WHO/UNICEF\nMeasles Initiative",
           x = 2001, y = length(country_order) + 0.5,
           hjust = -0.05, vjust = 1,
           size = 4, fontface = "bold", family = "Lato",
           color = bka_colors$title_dark) +
  coord_cartesian(clip = "off") +    # Prevent annotation text from being clipped
  # See Lesson 3.1 (Titles & Captions) — Minto-style title states the insight
  labs(
    title = "Mass immunization campaigns cut African measles cases by over 90%\nbut outbreaks persist where coverage gaps remain",
    subtitle = "Reported measles cases per million population by country, 1980-2023",
    x = NULL, y = NULL,
    caption = paste0(
      "Source: WHO Global Health Observatory (cases) & World Bank (population) | ",
      "Countries with 15+ years of reporting shown\n",
      "Original inspiration: r-graph-gallery.com/vaccination-heatmap.html"
    )
  ) +
  # See Lesson 2.2 (Custom Themes) — theme_bka() + heatmap-specific overrides
  theme_bka() +
  theme(
    panel.grid = element_blank(),    # Tufte: no gridlines for heatmaps
    axis.line = element_blank(),
    axis.text.y = element_text(size = 9, face = "bold", family = "Lato", hjust = 1),    # Few: readable with many rows
    axis.text.x = element_text(size = 8),
    legend.position = "bottom",
    legend.direction = "horizontal",
    legend.title = element_text(size = 9, face = "bold",
                                color = bka_colors$title_dark),
    legend.text = element_text(size = 8, color = bka_colors$subtitle_gray),
    plot.title = element_text(lineheight = 1.15),
    plot.margin = margin(15, 15, 25, 15)
  )

Key Insight

Real-world data tells a more complex story than synthetic data. The US measles heatmap shows a clean before/after break. Africa’s version reveals the same dramatic decline, but also the reality of periodic outbreaks when vaccination coverage slips — a pattern invisible in tidy synthetic datasets. Missing data (gray tiles) is itself informative, showing gaps in surveillance infrastructure.

Data Notes

What makes this chart different from the US version:

  1. Missing data is real. Gray tiles represent years where a country didn’t report to WHO — this is itself informative, showing gaps in surveillance infrastructure.

  2. Outbreaks recur. Unlike the US chart’s clean post-vaccine silence, Africa shows periodic red spikes even after 2001. These represent real outbreaks when vaccination coverage dips below herd immunity thresholds (~95% for measles).

  3. The rollout was gradual. Africa’s EPI (Expanded Programme on Immunization) began incorporating measles vaccine in the late 1970s-1980s, but mass supplementary campaigns didn’t accelerate until 2001. Some countries show early declines; others were late adopters.

  4. Per-capita normalization matters. Without dividing by population, Nigeria’s absolute case count would dominate the entire chart. Cases per million makes countries comparable. The data prep script (../data/prep_africa_measles.R) handles this join.

  5. Extreme outliers need squishing. Some country-years exceed 30,000 cases per million. oob = scales::squish maps these to the gradient maximum instead of showing them as gray NA tiles.

How the data was prepared

The CSV was generated by ../data/prep_africa_measles.R, which:

  1. Loads measles case counts from the WHO GHO API (jsonlite::fromJSON())
  2. Loads population from the World Bank API
  3. Joins on country code + year
  4. Computes cases_per_million = cases / population * 1e6
  5. Maps ISO3 codes to country names via the WHO dimension lookup
  6. Filters to countries with 15+ years of reporting

To refresh the data, re-run the prep script (requires internet and the jsonlite package).
